# 齐齐哈尔大学图书馆预约

## 一 项目介绍

齐齐哈尔大学图书馆自动预约程序,用python代码简单模拟人进行登陆,预约座位的过程!

## 二 使用说明
在目录下有一个accounter.text 目录
格式为:  
账号 密码 教室编号:座位编号1 座位编号2 座位编号3 ... 座位编号n 教室编号:教室编号:座位编号1 座位编号2 座位编号3 ... 座位编号n  
    如果有多个账户,写多行即可.只有预约到座位才会退出程序,所以请在第二天座位开放前几分钟运行程序.
